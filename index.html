CONTENEDOR FLEXIBLE

La propiedad display con valor flex o inline-flex nos permite establecer un contenedor flexible. Los hijos del contenedor flexible se llaman elementos flexibles.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
</div>
.contenedor{
   display: flex;
}

ESTABLECER LA DIRECCION DEL EJE PRINCIPAL DEL CONTENEDOR FLEXIBLE

La propiedad flex-wrap controla si el contenedor flexible es de una sola linea o de varias lineas y la direccion del eje secundario, que determina la direccion en la que se alinean las nuevas lineas.

RESUMEN: TERMINOS IMPORTANTES

ALINEAR LINEAS DEL CONTENEDOR FLEXIBLE

La propiedad align-content alinea las lineas de un contenedor flexible cuando hay espacio adicional en el eje secundario. Tenga en cuenta que esta propiedad no tiene efecto en un contenedor flexible de una sola linea.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-start;
}

Nota: Las líneas se colocan a partir del lado inicial secundario del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-end;
}

Nota: Las líneas se colocan a partir del lado final secundario del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: space-between;
}

Nota: Las líneas son distribuidas de manera uniforme en el contenedor flexible con respecto al eje secundario. El espaciado se hace de modo que la separación entre cualquier par de filas adyacentes sea el mismo. La primera linea se encuentra pegada al lado inicial secundario del contenedor flexible y la ultima linea se encuentra pegada al lado final secundario del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: center;
}

Nota: Las líneas se colocan al centro del contenedor flexible con respecto al eje secundario. El espacio entre el lado inicial secundario del contenedor flexible y la primera línea, es el mismo que el espacio que hay entre el lado final secundario del contenedor flexible y la última línea.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: space-around;
}

Nota: Las líneas son distribuidas uniformemente de modo que el espacio entre cualquier par de elementos adyacentes sea el mismo. El espacio antes de la primera línea y el espacio después de la última es igual a la mitad del espacio entre cualquier par de líneas adyacentes.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: stretch;
}

Nota: Las líneas son estiradas para usar el espacio sobrante

    En el caso de que el contenedor sea de 300px de alto y los elementos de 50px con excepción del segundo que es 100px de alto, tenemos:
  • La primera linea tiene 100px de alto
  • La segunda linea tiene 50px de alto
  • El espacio sobrante es 150px
    Este espacio sobrante se distribuye equitativamente entre las dos lineas:
  • La primera linea es ahora 175px de alto.
  • La segunda linea es ahora 125px de alto.

ELEMENTOS FLEXIBLES

MODIFICAR EL ORDEN DE LOS ELEMENTOS FLEXIBLES

Los elementos flexibles se disponen de forma predeterminada en el mismo orden en que aparecen en el documento de origen. La propiedad order se puede utilizar para cambiar este orden.

REGLA CSS

<div class="contenedor">
   <div class="flex-item-1">E1</div>
   <div class="flex-item-2">E2</div>
   <div class="flex-item-3">E3</div>
   <div class="flex-item-4">E4</div>
   <div class="flex-item-5">E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: nowrap;
   align-content: stretch;
   justify-content: flex-start;
   align-items: stretch;
}
.flex-item-1 { order: -3; }
.flex-item-2 { order:  0; }
.flex-item-3 { order: -5; }
.flex-item-4 { order: -2; }
.flex-item-5 { order: -4; }
.flex-item-6 { order: -1; }

Nota: Inicialmente cada elemento flexible tiene asociado un numero order igual a cero (0). El numero order de un elemento flexible puede modificarse usando la propiedad order. Los elementos se acomodan empezando por el que tiene el numero order mas bajo. Los elementos con el mismo order se presentan en el orden en que aparecen en el documento de origen.

TAMAÑO INICIAL

La propiedad flex-basis permite definir el tamaño de un elemento flexible a lo largo del eje principal del contenedor flexible.

Valores posibles:
  • auto [Valor por defecto]. Significa "observa mi propiedad width o height".
  • content. La palabra clave content significa "dimensionarlo según el contenido del elemento".
  • Una longitud (20%, 5rem, etc.)(

REGLA CSS

<div class="contenedor">
   <div class="flex-item-1">E1</div>
   <div class="flex-item-2">E2</div>
   <div class="flex-item-3">E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: nowrap;
   align-content: stretch;
   justify-content: flex-start;
   align-items: stretch;
}
.flex-item-1 { flex-basis: 100px; }
.flex-item-2 { flex-basis: 200px; }
.flex-item-3 { flex-basis: 100px; }

FACTOR DE CRECIMIENTO

La propiedad flex-grow especifica el factor de crecimiento de un elemento flexible, en su dirección principal. El factor de crecimiento determina cuánto crecerá el elemento flexible, en relación con el resto de elementos flexibles en el contenedor flexible cuando se distribuya el espacio libre del contenedor (El espacio libre es el tamaño del contenedor flexible menos el tamaño de todos los elementos flexibles juntos).

REGLA CSS

<div class="contenedor">
   <div class="flex-item-1">E1</div>
   <div class="flex-item-2">E2</div>
   <div class="flex-item-3">E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: nowrap;
   align-content: stretch;
   justify-content: flex-start;
   align-items: stretch;
}
.flex-item-1 {
   flex-basis: 150px;
   flex-grow: 1;
}
.flex-item-1 {
   flex-basis: 150px;
   flex-grow: 2;
}
.flex-item-1 {
   flex-basis: 150px;
   flex-grow: 1;
}

Nota: Si todos los elementos tienen flex-grow establecido en 1, el espacio restante en el contenedor se distribuirá por igual a todos los elementos flexibles. Si uno de los elementos flexibles tiene un valor de 2, este ocuparía el doble del espacio restante que los demás. Notar que cuando se omite, el valor por defecto es cero (0).

FACTOR DE CONTRACCION

La propiedad flex-shrink especifica el factor de contracción de un elemento flexible. Los elementos flexibles se encogerán para llenar el contenedor de acuerdo a su número flex-shrink, cuando el tamaño de los elementos flexibles sea mayor al del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div class="flex-item-1">E1</div>
   <div class="flex-item-2">E2</div>
   <div class="flex-item-3">E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: nowrap;
   align-content: stretch;
   justify-content: flex-start;
   align-items: stretch;
}
.flex-item-1 {
   flex-basis: 600px; 
   flex-shrink: 2;
 }
.flex-item-2 {
   flex-basis: 600px; 
   flex-shrink: 1; [Valor por defecto]
 }
.flex-item-3 {
   flex-basis: 600px; 
   flex-shrink: 2;
 }

ALINEAR ELEMENTOS FLEXIBLES A LO LARGO DEL EJE SECUNDARIO

La propiedad align-self permite que cada elemento flexible se puede alinear con respecto al eje secundario de la línea actual del contenedor flexible.

Valores posibles:
  • auto [Valor por defecto]. Se deja la responsabilidad de alineación al contenedor flexible.
  • flex-start
  • flex-end
  • center
  • stretch

REGLA CSS

<div class="contenedor">
   <div class="flex-item-1">E1</div>
   <div class="flex-item-2">E2</div>
   <div class="flex-item-3">E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: nowrap;
   align-content: stretch;
   justify-content: flex-start;
   align-items: flex-start;
}
.flex-item-1 {
   flex: 0 1 250px;
}
.flex-item-2 {
   flex: 0 1 250px;
}
.flex-item-3 {
   flex: 0 1 250px;
   align-self: flex-end;
}
