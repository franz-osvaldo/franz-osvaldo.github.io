CONTENEDOR FLEXIBLE

PROPIEDAD

La propiedad display con valor flex o inline-flex nos permite establecer un contenedor flexible. Los hijos del contenedor flexible se llaman elementos flexibles.

RESUMEN

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
</div>
.contenedor{
   display: flex;
}

ESTABLECER LA DIRECCION DEL EJE PRINCIPAL DEL CONTENEDOR FLEXIBLE

PROPIEDAD

La propiedad flex-direction permite establecer la direccion del eje principal del contenedor flexible.

RESUMEN

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row; [VALOR POR DEFECTO]
}
<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row-reverse;
}
<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: column;
}
<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: column-reverse;
}

CONFIGURAR EL CONTENEDOR FLEXIBLE PARA QUE SEA MULTI LINEA O DE UNA SOLA LINEA, Y ESTABLECER LA DIRECCION DE SU EJE SECUNDARIO

PROPIEDAD

La propiedad flex-wrap controla si el contenedor flexible es de una sola linea o de varias lineas y la direccion del eje secundario, que determina la direccion en la que se alinean las nuevas lineas.

RESUMEN

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: nowrap; [VALOR POR DEFECTO]
}
<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: wrap;
}
<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: wrap-reverse;
}

ALINEAR LINEAS DEL CONTENEDOR FLEXIBLE

PROPIEDAD

La propiedad align-content alinea las lineas de un contenedor flexible cuando hay espacio adicional en el eje secundario. Tenga en cuenta que esta propiedad no tiene efecto en un contenedor flexible de una sola linea.

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>

   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-start;

   <div>E4</div>

</div>

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
