CONTENEDOR FLEXIBLE

La propiedad display con valor flex o inline-flex nos permite establecer un contenedor flexible. Los hijos del contenedor flexible se llaman elementos flexibles.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
</div>
.contenedor{
   display: flex;
}

ESTABLECER LA DIRECCION DEL EJE PRINCIPAL DEL CONTENEDOR FLEXIBLE

La propiedad flex-direction permite establecer la direccion del eje principal del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row; [VALOR POR DEFECTO]
}

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row-reverse;
}

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: column;
}

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: column-reverse;

CONFIGURAR EL CONTENEDOR FLEXIBLE PARA QUE SEA MULTI LINEA O DE UNA SOLA LINEA, Y ESTABLECER LA DIRECCION DE SU EJE SECUNDARIO

La propiedad flex-wrap controla si el contenedor flexible es de una sola linea o de varias lineas y la direccion del eje secundario, que determina la direccion en la que se alinean las nuevas lineas.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: nowrap; [VALOR POR DEFECTO]
}

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: wrap;
}

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
  flex-wrap: wrap-reverse;
}

ALINEAR LINEAS DEL CONTENEDOR FLEXIBLE

La propiedad align-content alinea las lineas de un contenedor flexible cuando hay espacio adicional en el eje secundario. Tenga en cuenta que esta propiedad no tiene efecto en un contenedor flexible de una sola linea.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-start;
}

Nota: Las líneas se colocan a partir del inicio del eje secundario

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-end;
}

Nota: Las líneas se colocan a partir del final del eje secundario

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: space-between;
}

Nota: Las líneas son distribuidas de manera uniforme en el contenedor flexible. El espaciado se hace de modo que la separación entre cualquier par de elementos adyacentes sea el mismo. La primera linea se encuentra pegada al borde del contenedor flexible que coincide con el inicio del eje secundario y la ultima linea se encuentra pegada al borde opuesto del contenedor flexible.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: center;
}

Nota: Las líneas se colocan al centro del contenedor flexible con respecto al eje secundario. El espacio entre el borde del contenedor flexible que coincide con el inicio del eje secundario y la primera línea, y el que hay entre el borde opuesto del contenedor y la última línea es el mismo.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: space-around;
}

Nota: Las líneas son distribuidas uniformemente de modo que el espacio entre cualquier par de elementos adyacentes sea el mismo. El espacio antes de la primera línea y el espacio después de la última es igual a la mitad del espacio entre cualquier par de líneas adyacentes.

REGLA CSS

<div class="contenedor">
   <div>E1</div>
   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>
.contenedor{
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: stretch;
}

Nota: Las líneas son estiradas para usar el espacio sobrante

    En el caso de que el contenedor sea de 300px de alto y los elementos de 50px con excepción del segundo que es 100px de alto, tenemos:
  • La primera linea tiene 100px de alto
  • La segunda linea tiene 50px de alto
  • El espacio sobrante es 150px
    Este espacio sobrante se distribuye equitativamente entre las dos lineas:
  • La primera linea es ahora 175px de alto.
  • La segunda linea es ahora 125px de alto.

   <div>E2</div>
   <div>E3</div>
   <div>E4</div>
   <div>E5</div>
</div>

   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   align-content: flex-start;
  justify-content: flex-start; [VALOR POR DEFECTO]
}
